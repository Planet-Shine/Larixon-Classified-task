Как смотреть результат?

/frontend/index.html  — файл, который нужно запустить.

Возможные команды в консоли из директории проекта:
npm run compile — собирает вновь статику проекта.
npm run server — запустит сервер, после чего приложение
можно будет посмотреть в браузере по адресу http://127.0.0.1:8080/frontend/index.html.
